# Model Information

This folder is intended to store trained machine learning models generated during the project.

Due to file size limitations, trained model files are not included in this repository.

## Models Evaluated

The project evaluates the following machine learning approaches:

### Traditional Machine Learning Models

* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest
* XGBoost

### Deep Learning / Transformer Models

* Neural Network classifier
* DistilBERT
* RoBERTa

## Model Outputs

Trained models may include:

* Text vectorizers (TF-IDF)
* Classification models
* Transformer checkpoints
* Model evaluation artifacts

## Storage Location

For reproducibility, trained models are stored externally and can be generated by running:

```
notebooks/03_model_training.ipynb
```

The trained models are used for evaluation in:

```
notebooks/04_evaluation.ipynb
```

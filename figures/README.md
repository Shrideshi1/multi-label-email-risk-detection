# Figures

This folder contains visualizations and figures generated during the **Multi-Label Email Risk Detection** project.

## Contents

Figures may include:

* Dataset distribution plots
* Label frequency visualizations
* Class imbalance analysis
* Model performance comparisons
* Confusion matrices
* Evaluation metric charts
* Feature importance visualizations

## Generation

Figures are generated from the analysis and evaluation notebooks:

```text
notebooks/
├── 02_feature_engineering.ipynb
└── 04_evaluation.ipynb
```

## Storage

Generated figures are stored here to support project documentation, analysis, and reporting.
